# HigherCareer

HigherCareer is a comprehensive Primary School Management System designed to streamline and simplify school administration tasks. This system provides essential features to manage students, parents, teachers, classes, subjects, fees, and generate reports efficiently. It is tailored to meet the needs of primary schools with a focus on usability and effectiveness.

---

## Features

### 1. Admin Dashboard
   - Centralized interface for managing school operations.
   - Provides quick access to key metrics and functionality.

### 2. Student Management
   - Enroll new students and maintain student records.
   - Track student performance, attendance, and activities.

### 3. Parent Management
   - Maintain parent details, ensuring easy communication.
   - Link parents to their children for seamless information sharing.

### 4. Teacher Management
   - Manage teacher profiles, assignments, and schedules.
   - Keep track of teacher performance and feedback.

### 5. Class and Subject Management
   - Organize classes and subjects effectively.
   - Assign teachers and students to classes with ease.

### 6. Fee Management
   - Manage fee structures, payments, and receipts.
   - Generate financial reports for better transparency.

### 7. Report Generation
   - Generate detailed reports for students, teachers, and school activities.
   - Export reports for administrative purposes.

---

## Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (Express Framework)
- Database: PostgreSQL
- Hosting: AWS with DuckDNS and HTTPS via Caddy
- Design: Implements a modern UI using a custom color palette and gradient backgrounds.
